# Debian LAMP WordPress Setup Script

This bash script automates the installation of the LAMP stack (Apache, MariaDB, PHP) on Debian 9 and configures WordPress. It sets up the database, downloads WordPress, and configures Apache's virtual host for basic access.

## Language

`bash`

## How to Run

1. Save the code as `setup_lamp_wordpress.sh`.
2. Make it executable: `chmod +x setup_lamp_wordpress.sh`.
3. Run with root privileges: `sudo ./setup_lamp_wordpress.sh`.
